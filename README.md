## Installation

```sh
npm install -g @coderosh/myriad
```

## Usage

```sh
# Open repl
myriad

# Run file
myriad file.mainl
myriad doc.genzl
myriad fiwe.uwul
myriad scroll.piratel
myriad file.nepalil

# Change language in repl
> lang
# > lang
# > Which ? genz
```
