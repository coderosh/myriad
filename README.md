## Installation

```sh
npm install -g @coderosh/myriad
```

## Usage

- repl
  ```sh
  myriad
  ```
- run file

  ```sh
  # main
  myriad name.mainl

  # genz
  myriad name.genzl

  # uwu
  myriad name.uwul

  # pirate
  myriad name.piratel

  # nepali
  myriad name.nepalil
  ```
